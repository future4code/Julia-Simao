.descrição do projeto {


O projeto proposto era o desenvolvimento de um site
para uma loja. O setor escolhido foi o alimentício,
conseguindo mostrar através da página uma loja de cookies
gourmet. O site conta com um menu inicial, controle de estoque
aba para adicionar produtos e a lista de funcionários.
Foi utilizado HTML e CSS para o desenvolvimento.
